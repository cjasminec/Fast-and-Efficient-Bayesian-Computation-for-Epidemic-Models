# Fast-and-Efficient-Bayesian-Computation-for-Epidemic-Models
This repository contains all of the code used to generate graphs and figures for my dissertation, as well as the algorithms implemented throughout.

This repository is part of the work I have done on Bayesian Inference for epidemic models as part of a third year undergraduate dissertation. It covers deterministic epidemic models for both the SIR and the SEIR model, as well as the stochastic Poisson leap model. It also focuses on Bayesian methods for inference, specifically the Metropolis-Hastings algorithm from Markov chain Monte Carlo, and approximate Bayesian computation (ABC).

The datasets used in this project were from a case of influenza in a boys' boarding school (SIR), and an outbreak of Ebola in Western Africa (SEIR). These are also attached in the repository.
