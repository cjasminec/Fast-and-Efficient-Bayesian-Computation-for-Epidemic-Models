# Fast-and-Efficient-Bayesian-Computation-for-Epidemic-Models
This contains all of the code used to generate graphs and figures for my dissertation, as well as the algorithms implemented throughout.
This repository is part of the work I have done on Bayesian Inference for epidemic models, as part of a third year undergraduate dissertation. It covers deterministic epidemic models (specifically from the SIR model), elements of both frequentist and Bayesian methods, though I focus in much greater depth on the Bayesian framework.
